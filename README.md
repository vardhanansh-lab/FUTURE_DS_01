**# Retail Sales Performance Dashboard**



This project analyzes retail sales data using Power BI.



**## Key Insights**



\- Total sales are approximately $12.6M.

\- Technology category generates the highest revenue.

\- Central region contributes the most sales.

\- Phones and Copiers are the best-performing sub-categories.

\- Apple Smart Phone Full Size is the top-selling product.
